# 家計簿（ヘルパー）/ Household Ledger — PWA

レシートを読み取って収支を管理する、独立したWebアプリ（PWA）です。
Claude のアーティファクトではなく、自分のサイトとして動くので、データ保存と読み取りが普通に使えます。

## 含まれるファイル

| ファイル | 役割 |
|---|---|
| `index.html` | アプリ本体 |
| `manifest.webmanifest` | ホーム画面追加・アプリ名・アイコン定義 |
| `service-worker.js` | オフライン動作（アプリ本体をキャッシュ） |
| `icon-192.png` / `icon-512.png` | アプリアイコン |
| `apple-touch-icon.png` | iOSホーム画面用アイコン |

すべて同じフォルダ（リポジトリのルート）に置いてください。

## GitHub Pages で公開する手順

1. GitHub にログイン → 右上「＋」→ **New repository**
   - Repository name: 例 `kakeibo`
   - **Public** を選択 → Create repository
2. 作成したリポジトリで **Add file → Upload files**
   - このフォルダ内の全ファイルをドラッグ＆ドロップ
   - 下の **Commit changes** を押す
3. **Settings → Pages**
   - Source: **Deploy from a branch**
   - Branch: **main** / **/ (root)** → **Save**
4. 1分ほど待つと URL が表示されます
   - `https://<あなたのユーザー名>.github.io/kakeibo/`
5. スマホでその URL を開く → 共有メニュー → **ホーム画面に追加**
   - これでアプリのように起動できます（オフラインも可）

## 使い方

- **スキャン → 画像を読み込む**：レシート写真を選ぶと自動で明細を読み取り → 確認して保存
- **ホーム → ＋ お金を渡す**：ヘルパーに渡した現金を記録
- **ホーム**：手元残高（渡した − 使った）と今月の収支・カテゴリ内訳
- **履歴**：入出金の通帳・削除
- **設定**：予算・通貨・カテゴリ・バックアップ・**読み取り方式**

## 読み取り方式の切り替え（設定 → レシート読み取り）

- **無料・オフライン（既定）**：端末内の Tesseract.js で読み取り。キー不要。初回だけエンジンを取得するため通信が必要。精度は中程度。
- **高精度（自分のClaude APIキー）**：`console.anthropic.com` で取得したキーを入力すると、Claude のビジョンで高精度に読み取ります。
  - キーは**この端末のブラウザ内だけ**に保存されます（外部送信なし。読み取りリクエスト時のみ Anthropic API に使用）。
  - 公開リポジトリにキーを書かないでください。入力はアプリの設定画面からのみ行います。
  - 従量課金（少額）が発生します。

## データについて

- 記録はブラウザの localStorage に保存されます（この端末内のみ）。
- 画像は保存されません。読み取った明細データだけが残ります。
- バックアップ：設定 → **JSONバックアップ**で退避、**復元**で読み込み。Excel用に **収支CSV / 明細CSV** も出力できます。
- 端末を変える・ブラウザのデータを消す前には、JSONバックアップを取ってください。

## カスタマイズ（開発を続ける場合）

- UI・ロジックはすべて `index.html` の1ファイルにまとまっています（外部依存は Web フォントと Tesseract.js のみ）。
- カテゴリの初期値は `DEFAULT_CATS`、文言は `I18N`（en / ja）にあります。
- 読み取りは `extractReceipt()` が入口で、`extractWithTesseract()` と `extractWithClaude()` に分かれています。
